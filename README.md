Booking Lap Badminton

Aplikasi desktop Java berbasis Swing untuk melakukan booking lapangan badminton dengan sistem jadwal realtime menggunakan database MySQL dan konsep MVC.

Fitur
Booking lapangan badminton
Cek jadwal realtime
CRUD data lapangan
Menampilkan data booking
Login admin dan register
Validasi bentrok jadwal
Pencarian lapangan
Perhitungan otomatis total harga
Status lapangan (tersedia / maintenance)
Auto refresh jadwal realtime
